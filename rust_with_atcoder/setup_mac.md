## Rustのインストール
1. 下記コマンドを実行。

    1~3の選択を求められるので、1を入力してEnter。
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

2. バージョン確認

    シェルを再起動してcargoのバージョンを確認できればRustのインストール完了！
    ```bash
    cargo version
    ~~~ 実行結果
    cargo 1.95.0 (f2d3ce0bd 2026-03-21)
    ```
